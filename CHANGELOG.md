# Changelog

## v1.0.3 - 2026-06-22

- 精简设置页，移除软件更新和数据源区块。
- 调整设置页顺序为基金操作提醒、自动刷新、主弹窗。
- 精简基金详情顶部收益卡，暂时只展示当日涨幅。

## v1.0.2 - 2026-06-22

- 将更新提示栏移动到顶部收益区，并在显示更新栏时隐藏顶部更新按钮。
- 优化更新检查逻辑，仅在存在可自动安装的 zip 包时提示可更新。
- 移除顶部刷新状态右侧的空更新按钮占位。

## v1.0.1 - 2026-06-22

- 优化主弹窗顶部三角与面板背景衔接，减少颜色断层。
- 调整首页工具栏按钮、基金标签和收益率展示样式。
- 优化基金详情页当日涨幅重点展示和持仓指标排版。
- 支持在设置页直接输入主弹窗高度，并保留滑动调节。
- 优化 Swift 版 DMG 打包布局。

## v0.0.1 - 2026-06-19

- 重置项目版本，从当前基金持仓工具重新开始维护。
- 接入东方财富/天天基金相关基金数据源，保留基金增加、持仓、收益统计等核心流程。
- 移除股票、货币、指数、板块、榜单、功能中心、钱包管理等非核心入口。
- 重做弹窗首页浅色/深色布局，优化顶部收益区、工具条、基金列表和详情快照。
- 精简设置页和右键菜单，更新日志改为读取本项目本地 Changelog。
